WaitForChild

WaitForChild is a set of functions that waits for descendants conditionally.

Functions​

byPredicate​

</>
WaitForChild.byPredicate(
root: Instance,
timeout: number,--

How long before the promise automatically resolves as nil.

predicate?: (Instance) → boolean--

The conditions under which the search has successfully found a result.

) → Promise<Instance?>--

The result of the search wrapped as a promise.

First looks through the children of root for an Instance that passes the conditions of the predicate and if none are found begins listening for new instances and changes to existing ones. 

local found = byPredicate(workspace, math.huge, function(child)
	-- waiting for the first descendant of workspace that's a BasePart and has a certain color
	return child:IsA("BasePart") and child.Color = Color3.fromRGB(100, 120, 80)
end):expect()
